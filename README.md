# 🌸 Iris Data Exploration Project

## 📊 Task 2 - Data Science Internship

This project explores the Iris dataset using Python.

## 🔍 Work Done:
- Loaded dataset using Seaborn
- Checked dataset structure
- Verified no missing values
- Generated summary statistics
- Visualized data using pairplot, histogram, and boxplot

## 📌 Insights:
- Setosa is clearly separable
- Versicolor and Virginica overlap slightly
- Petal features are most useful

## 🛠 Tools Used:
- Python
- Pandas
- Seaborn
- Matplotlib
